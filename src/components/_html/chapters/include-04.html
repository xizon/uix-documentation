4. Import Demo

Once you have installed all theme Plugins you can import the Demo Content. By installing the demo content, your theme will look like the one you see on our demo. Restroe available content more than 95%. This action will let you understand how the theme works by allowing you to modify a content that is already there rather than creating it from scratch.


Step 1.
Go to Dashboard » Tools » Import in the admin panel.

Step 2.
Click on "WordPress", this will install the WordPress Importer plugin.

Step 3.
Choose the Demo-Content/shadowerpro_wordpress_demo_content.xml downloaded in the .zip above. Once selected, Click "Upload file and import".

On the next page you will be asked to assign a user to the posts you are importing. Make sure you have download and import file attachments checked. Click the "Submit" button. This will take a little while to upload.
